# YT Stream 🔴

**फक्त तुमच्यासाठी** — Screen + Internal Audio → YouTube Live

## Features
- ✅ Screen capture
- ✅ Internal audio (phone चा आतला आवाज)
- ✅ YouTube RTMP ला direct stream
- ✅ No watermark, No time limit, Completely free
- ✅ Android 10+ (API 29+)

## APK Install करायला
1. वरती **Actions** tab → latest build → **ytstream-apk** download करा
2. Phone वर install करा (Unknown sources allow करा)
3. Done!

## वापरायला
1. App उघडा
2. **YouTube Studio** → Go Live → Stream tab → **Stream Key** copy करा
3. App मध्ये Stream Key paste करा
4. **START STREAM** दाबा
5. Screen capture permission द्या
6. झालं! 🔴

## Stream Key कुठे मिळेल?
`studio.youtube.com` → **Go Live** → **Stream** tab → Stream Key
